# Proyecto: Linaje Valderrey

Este proyecto es una página web conmemorativa y genealógica dedicada al apellido **Valderrey**. El objetivo es documentar la historia, heráldica y distribución global de este linaje de origen leonés.

## Contexto Histórico
- **Origen:** Toponímico, de la villa de Valderrey en la Maragatería (León, España).
- **Significado:** Proviene de "Val de Rey" (Valle del Rey), indicando jurisdicción real.
- **Hito Histórico:** Participación en la defensa de San Quintín (1557) bajo el mando del Duque de Feria.
- **Heráldica:** Escudo partido. 1º Sinople (verde) con torre de oro. 2º Azur (azul) con puente de plata sobre ondas.

## Estructura del Proyecto
- `index.html`: Estructura y contenido con animaciones AOS.
- `styles.css`: Diseño premium, paleta oro/negro/verde/azul, tipografías Cinzel y Playfair Display.
- `Escudo_de_Valderrey.svg.png`: Imagen principal del blasón familiar.
- `651698024_927513593258519_1498279591937623386_n.jpg`: Fotografía actualizada de Piedad Valderrey.
- `CNAME`: Configuración para el dominio personalizado `valderrey.com`.

## Tecnologías y Hosting
- **Frontend:** HTML5, CSS3, animaciones AOS (Animate On Scroll).
- **Visualización de Datos:** Gráficos de barras CSS personalizados para la distribución provincial basados en datos del INE.
- **Hosting:** GitHub Pages con dominio personalizado en Cloudflare.

## Instrucciones para el Crecimiento del Proyecto
1. **Estética:** Mantener el estilo "Premium/Heritage". Usar degradados dorados y tipografías serif clásicas.
2. **Interactividad:** Seguir utilizando AOS para las entradas de secciones.
3. **Datos:** Cualquier nueva sección de datos (como la distribución geográfica) debe basarse en fuentes oficiales como el INE. Actualmente se utiliza una cuadrícula de infografía provincial en lugar de un mapa SVG para mayor claridad.
4. **Despliegue:** Cada cambio debe ser subido a la rama `main` de GitHub para actualizar la web en vivo.
